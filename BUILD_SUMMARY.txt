HOW TO PLAY
─────────────────────────────────────────────────────────────────────────────
1. Open index.html in a browser (or use serve.sh for local server)
2. Pick a skill from the sidebar
3. Click "Start" to begin an action
4. Watch progress bars fill up
5. Gain XP and loot upon completion
6. Level up skills to unlock harder activities
7. Combine skills for synergies
8. Go AFK and come back to offline progress

SAVE SYSTEM
─────────────────────────────────────────────────────────────────────────────
  Auto-save:   Every 30 seconds (automatic, no action needed)
  Manual save: Click "💾 Save" button in header
  Export:      Click "📥 Export Save" → code copied to clipboard
  Import:      Click "📤 Import Save" → paste your save code
  Reset:       Click "⚠️  Reset Game" → start fresh

GETTING STARTED
─────────────────────────────────────────────────────────────────────────────
  OPTION 1 (Direct):
    1. Double-click index.html
    2. Game starts immediately!

  OPTION 2 (Server - Recommended):
    1. Run: ./serve.sh (Mac/Linux) or python -m http.server 8000 (Windows)
    2. Open: http://localhost:8000
    3. Game runs from local server

  OPTION 3 (Node.js):
    1. Run: npx http-server -p 8000
    2. Open: http://localhost:8000
